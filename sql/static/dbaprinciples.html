. 变更申请页面: 
	1. 支持insert ... values, insert ... select 和 update ... set三种语法，update语句必须带有where条件
	2. sql提交后会估算影响行数，作为审核参考，影响行数较大的会重点审核
	3. sql提交框里支持输入多个sql，用";"隔开
	4. 如果一个工单包含多条语句,自动执行时,只有所有语句全部执行成功才会commit，否则会全部回滚。
. 查询页面:
	1.支持select语句
	2.查看表结构:desc drawcenter.t1
	3.查看索引:show index from drawcenter.t1
	4.限制返回的总行数，仅作为业务数据校验使用
. 语句中的表名必须带有schema前缀，比如写成insert into drawcenter.t1(id) values (1); 而不是insert into t1(id) values (1);
. comment语句解析会报错，dba确认无误后也可以自动或手工执行
